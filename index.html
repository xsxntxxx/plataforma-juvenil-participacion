Plataforma Juvenil de Participación

Proyecto académico para aprender participación juvenil, transparencia, control ciudadano y trazabilidad mediante herramientas digitales.

Propósito del proyecto

Esta plataforma se construirá progresivamente como ejercicio pedagógico para comprender cómo la tecnología puede apoyar procesos comunitarios de participación y organización.

Advertencia pedagógica

Este proyecto no corresponde a una elección real, no recolecta datos personales reales y no debe utilizarse para hacer acusaciones.
